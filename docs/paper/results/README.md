# Results

This directory holds the benchmark outputs behind the paper: aggregate scores,
certificates, and summary JSONs. It does **not** include raw per-instance
prediction dumps (`predictions/*.jsonl` from the long-horizon SWE-bench runs)
or the full per-instance longhorizon result file — those ran well past 1 MB
each and aren't useful without the harness that produced them.

Raw predictions are available on request (open an issue) or as release
artifacts attached to the paper's tagged release.
